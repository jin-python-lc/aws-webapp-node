試験1 - 分野別

移行・移転

＜第10問＞
ある企業では、70TBの自社業務データを保存しているオンプレミスのファイルサーバーを所有しています。その業務データのバックアップ先としてS3 Glacierへのデータ移転を検討しています。企業のネットワーク回線の負担を減らしつつ、費用対効果の高い移転方法として適切なものを選択してください。

答え：3

1. S3へWebブラウザからファイルをアップロードする。S3上に移転されたデータをGlacierにアーカイブするライフサイクルポリシーを適用する。

2. VPNで企業のオンプレミスサーバとAWSを接続し、VPN経由でGlacierにデータをアップロードする。

3. Snowballを利用してS3バケットへデータを移転し、S3上に移転されたデータをGlacierにアーカイブするライフサイクルポリシーを適用する。

4. VPNで企業のオンプレミスサーバとAWSを接続し、VPN経由でGlacierにデータをアップロードする。S3上に移転されたデータをGlacierにアーカイブするライフサイクルポリシーを適用する。


・1-企業のネットワーク回線の負担を軽減するという要件があるため、70TBのデータをインターネットやVPN経由で転送する選択肢は不正解です。

・2-企業のネットワーク回線の負担を軽減するという要件があるため、70TBのデータをインターネットやVPN経由で転送する選択肢は不正解です。

・3-Snowballは、S3へデータをすばやく一括転送します。接続制限、帯域幅の制約、ネットワーク接続コスト、レガシー環境に起因する問題などが存在する場合や、データの収集が遠隔地で行われる場合にデータを転送するのに役立ちます。snowballの容量は80TBです。

・4-企業のネットワーク回線の負担を軽減するという要件があるため、70TBのデータをインターネットやVPN経由で転送する選択肢は不正解です。

＜第17問＞
オンプレミスのストレージからAWSのストレージへと移行を計画しています。既存のストレージは複数台のオンプレミスサーバーから利用されることが多く、さらに高いI/O性能が求められるため、EFSを利用することとしました。EFSへファイルデータを効率的かつ高速に移動するために利用するべきサービスを選択してください。

答え：1

1. DataSync

2. AWS Backup

3. Storage Gateway

4. SMS


・1-DataSyncは、オンプレミスストレージとEFS間でデータを迅速かつ簡単に移動することができるマネージド型のデータ転送サービスです。オープンソースツールと比較すると最大10倍の速度で、AWS Direct Connectまたはインターネット経由で転送可能です。

・2-AWS バックアップはEFS ファイルシステムの中央管理とバックアップの自動化が簡単にできるフルマネージド型のバックアップサービスです。EFSのデータ移行には利用できません。

・3-Storage Gatewayもオンプレミスからのデータ連携が可能ですが、EFSへの移行は対応していません。対応しているのは、S3、S3 Glacier、S3 Glacier Deep Archive、FSx for Windows、EBS、AWS Backupなどです。

・4-Server Migration Serviceは、オンプレミスの VMwareのvSphere、MicrosoftのHyper-V/SCVMM、Azure仮想マシンのAWSへの移行を自動化します。EFSのデータ移行には利用できません。
